Smart Pump Controller











Key Features:

Core Functionality
  • Automatic ON/OFF Control : Operates the water pump automatically based on
          tank water levels or preset schedules.
  • Manual Override Mode : Allows manual control via buttons, Bluetooth app, or physical switch.
Time-Based Scheduling (RTC Integrated)
  • Real-Time Clock (DS3231) : Ensures precise time-based operations even during power failures.
  • Multiple Schedules : User can set start and stop times for daily or weekly pump operations.
  • EEPROM Memory Storage : Retains all user settings and schedules even after power loss.
Safety & Automation
  • Dry Run Protection : Detects when there’s no water in the source and automatically stops the pump.
  • Tank Overflow Prevention : Stops pump when the tank is full using float sensors.
User Interface
  • LCD Display (16x2 or 20x4) : Shows time, status, and operation modes clearly.
  • Push Buttons / Keypad : Easy local setup of schedules and modes without needing a phone.
Power Management
  • Low-Power Standby Mode : Minimizes power consumption when idle.
  • Relay-Based Pump Control : Reliable and electrically isolated switching for motor operation.
