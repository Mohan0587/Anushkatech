Master Bell System

Key Features:
  • Automatic Daily Bell Scheduling.
  • Supports up to 20 programmable bell times per day.
  • Each bell can play a unique MP3 track from USB or SD card.
  • Track selection via track code (001 to 099) matching audio file name (e.g., 001.mp3).
  • User-defined repeat count for each bell (minimum 1 repeat).
  • Real-Time Clock (RTC)
  • Built-in DS3231 RTC (Real-Time Clock) for accurate and consistent timekeeping,
           even during power failures.
  • Stores all bell times, track codes, and settings permanently-no data loss on power off.
  • Easy navigation using UP, DOWN, LEFT, RIGHT buttons.
  • Settings displayed on 16x2 LCD screen with blinking area for editing.
  • Built-in Audio Amplifier.
  • External speakers (SPK1 & SPK2 terminals).
  • Optional line-out for external amplifier connection.
  • USB/SD Card Compatible.
  • Supports FAT32-formatted USB drives/SD cards, plug & play.
  • Manual Control Option.
  • Optional manual trigger for instant bell or test play.
  • Compact, wall-mountable enclosure with power safety feature.

Technical Specification:

           Power Supply: 220V AC input (internal 24V DC, 2.5A regulated for amplifier)
            Bell Output:Audio output via built-in amplifier (up to 50W total)
            Operating Temperature:0°C to 50°C
            Dimensions:[8X6X2.5]inches
            Weight:[400]gram
